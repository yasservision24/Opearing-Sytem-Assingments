1-to compile the kernel_mod run the command in terminal ->> make
2-to load the module in kernel run the ->> sudo insmod kernel_mod.ko
3-to list all the kernel module run the ->>lsmod
4- now run ->>sudo dmesg
5- to unload the mod in kernel run the ->> sudo rmmod kernel_mod
6-now can check the kernel by -> lsmod   OR  ->> sudo dmesg
